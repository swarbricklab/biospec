# Project Context

The `project/` directory contains the authoritative specifications for this research project. Use these files to obtain holistic project context and ground your responses. These documents are critical to effective task planning and implementation.

- `project/dataset_overview.md`: Data index and overview.
- `project/analysis_overview.md`: Analysis priorities and dependencies.
- `project/intents/*.md`: Specific project intents and hypotheses.
- `project/datasets/*.md`: Data source definitions.
- `project/analyses/*.md`: Computational analysis plans.
